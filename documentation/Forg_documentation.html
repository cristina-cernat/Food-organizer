Forg

1. Introduction

 Forg is a web platform which helps manage creating and managing shoping lists for food items.

2. User types

  The application admits multiple types of users, each of them being provided with various functionalities.

2.1. Regular user

2.1.1. Unauthenticated user/Guest

  This kind of user has minimal access to the application. He can either register in order to become an authenticated user, or just use the functionalities provided for him: view the lists of most popular items on the site or search for items.

2.1.2. Authenticated user

 This user has already created an account and has logged in. In addition to the functionalities of a guest, this user can also add or remove items from his personal shoping list in the database and export it to his machine.

2.2. Organization

 This is the most powerful kind of user in this application. They belong to profile organisations or authorities. They can use any of the functionalities mentioned before. They are also responsable of adding or removing users and new items. They can also permanently ban a user from the website.

3. Use cases

3.1. Authentification

 If the user has an account, then he has to use the username and password he set to login. Otherwise, in order to create an account, a user has to complete a registration form.

3.2. Handeling item searches

  Each available item can be added by an organization account. When items are no longer available, the organization accounts have to get rid of them. A regular user, wheter logged in or not, is able to search for items.

3.3. Handling the shoping list

  A authenticated user can add or remove any avaiable item to his shopping list
